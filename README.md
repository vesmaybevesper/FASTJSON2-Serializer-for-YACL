<p>Am I Up To Date is a lightweight mod designed to be included in modpacks to check the local version against the most recent version and notify the user of an update.</p>

---------------------------------------------------------------------------------------------------
## Usage

To get the version API URL for your modpack:

1. Go to https://docs.modrinth.com/api/operations/getprojectversions/
2. Expand the "GET" collapseable panel
3. In the "Production server" section, replace the {id|slug} with your Modpack ID.

   _This can be found on your modpack page via the more options button -> "Copy ID"_

**Example (for the "PyPack" modpack): https://api.modrinth.com/v2/project/olD3AEfS/version**

(Usage guide written by WimpieRatte on GitHub)
